## 2. 주요 화면

### 2.1 메인 차트 뷰

- **좌측 사이드바:** 관심종목 리스트 (그룹별)

## 3. 컴포넌트

### 3.1 차트 컴포넌트

- `<CandlestickChart />` - 메인 캔들 차트
- `<VolumeChart />` - 거래량 차트
- `<IndicatorChart />` - RSI, MACD 등 서브 지표
- `<DrawingLayer />` - 작도 오버레이

### 3.2 사이드바 컴포넌트

- `<WatchlistPanel />` - 관심종목 패널
- `<WatchlistGroup />` - 그룹 (접기/펴기)
- `<StockItem />` - 개별 종목 (가격, 등락률)

### 3.3 툴바 컴포넌트

- `<TimeframeSelector />` - 1m, 5m, 1h, 1D...
- `<IndicatorToggle />` - 지표 on/off
- `<DrawingTools />` - 작도 도구 선택

### 3.4 모달 컴포넌트

- `<SearchModal />` - 종목 검색
- `<IndicatorSettings />` - 지표 파라미터
- `<AlertSettings />` - 알림 조건

## 4. 인터랙션

### 차트

- **줌:** 마우스 휠
- **팬:** 드래그
- **크로스헤어:** 마우스 이동
- **툴팁:** 호버 시 OHLCV 표시

### 작도

- **추세선:** 시작점 클릭 → 끝점 클릭
- **수평선:** 클릭한 Y 위치에 수평선
- **편집:** 기존 작도 클릭 → 핸들로 조절
- **삭제:** 선택 후 Delete 키

### 알림

- **빠른 알림:** 차트에서 우클릭 → "이 가격에 알림"
- **상세 설정:** 알림 패널에서 조건 조합

## 6. 반응형

### Desktop (1200px+)

- 전체 레이아웃 표시
- 사이드바 고정

### Tablet (768px - 1199px)

- 사이드바 접기 가능
- 툴바 아이콘만

### Mobile (768px 미만)

- MVP에서는 미지원
- 추후 별도 모바일 뷰

## 7. 접근성

- 키보드 네비게이션
- 고대비 색상
- 스크린리더 라벨
- 단축키 지원 (T: 추세선, H: 수평선 등)
